password-revealer.js

Easily reveal/hide passwords in input fields.

npm install password-revealer --save

Live Examples

Init with default behaviour

PasswordRevealer('#password-field').init()

Init with password revealed by default

PasswordRevealer('#password-field', {
  isRevealed: true
}).init()

Init with custom trigger

PasswordRevealer('#password-field', {
  trigger: {
    selector: 'input[type="checkbox"]'
  }
}).init()
Toggle password

Show pragmatically

PasswordRevealer('#password-field').show()
